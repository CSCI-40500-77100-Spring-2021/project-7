# STONKS Prototype

## Instructions
1. Clone Bermet's branch
2. Install in the terminal:
- npm i
- npm i canvasjs-react-charts
- npm i axios
- npm i axios-cache-adapter
- npm i localforage 
3. After you install everything go to localhost:3000 and enter stock ticker
